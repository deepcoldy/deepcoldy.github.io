昨晚兄弟说不会写轮播，我说这不简单，其实就是图片的位移。

他说，对，但我不会写。

然后我就随便搞了一段出来。

在这记录一下，然后删掉把，没啥技术水平。

主要就是用绝对定位将两张图片定位到一个div里，设置好div的大小只与一张图片相等，然后overflow:hidden，紧接着就改变两张图的位置就好了。代码如下：

注意setInterval的用法，第一个方法是要加引号的。

```
<!DOCTYPE html>

<html>

<head>

   <title></title>

   <style type="text/css">

      body{

         margin: 0px;

         padding: 0px;

      }

      .con{

         margin: 10px;

         width: 500px;

         height: 300px;

         overflow: hidden;

         position: relative;

      }

      #img1{
         width: 500px;
         height: 300px;
         position: absolute;
         left: 0px;
      }

      #img2{
         width: 500px;
         height: 300px;
         position: absolute;
         left: 500px;
      }
   </style>
   <script type="text/javascript">
      var i=0;
      var j=0;
      var k=50;
      function time(id){
         move('img1');
         move2('img2');
         var int=setInterval("time('id')",50);
      }
      function move(id){
         var pic=document.getElementById(id)
         var dis=pic.style.left;
         if(dis == "-500px"){
            window.clearInterval(int);
         }
         j--;
         dis=j+"0px";
         pic.style.left=dis;
      }
      function move2(id){
         var pic=document.getElementById(id)
         var dis=pic.style.left;
         if(dis == "0px"){
            window.clearInterval(int);
         }
         k--;
         dis=k+"0px";
         pic.style.left=dis;
      }
   </script>
</head>
<body onload="time('img1')">
<div class="con">
   <img class="img" id="img1" src="1.jpg">
   <img class="img" id="img2" src="2.jpg">
</div>
<input type="button"  value="move">
</body>
</html>
```
