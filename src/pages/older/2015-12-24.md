#### 问题：在WordPress中使用裁剪图片功能时，出现:”在裁剪您的图像时发生了错误。”或者”There has been an error cropping your image.”

原因：缺少PHP GD库。


Ubuntu下运行：

> sudo apt-get install php5-gd

CentOS下运行：

> yum install php-gd

安装PHP-GD库后重启httpd（Apache）服务即可使用~
